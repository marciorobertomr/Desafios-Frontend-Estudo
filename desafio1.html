Entendendo Comunicação Client x Server

História da Web

Voltar

Um pouco de história: Os computadores surgiram em meados da segunda guerra e funcionavam por meio de circuitos e válvulas eletrônicas, sendo de Primeira Geração. Tinham o uso bem restrito e eram enormes, além de consumirem muita energia. Na Segunda Geração, eles diminuiram um pouco e tiveram as válvulas substituídas pelos transistores, permitindo que fossem comercializados. A Terceira Geração marcou com a mudança o funcionamento para circuitos integrados, o que reduziu drasticamente o seu tamanho e permitiu o produção de computadores pessoais. Na Quarta Geração, que é a que vivemos hoje, os computadores se poupularizaram ainda mais e passaram a ter uma capacidade de processamento poderosa.

O que são Clients e o que são Servers ?

Voltar

Nesse módulo, foi passado uma explicação que diferenciou os tipos de aplicações, linguagens e equipamentos. Aqui, foi apresentado os aspectos das aplicações voltadas para os clientes (Clients) e para os servidores (Servers). De forma geral, aplicações voltadas para o cliente possuem o aspecto visual com o qual o cliente interage, como sites, navegadores, aplicativos de celulares. Já os servidores é onde ficam armazenadas as informações, acessíveis através de uma programação back-end, que retorna informações para o front-end mostrar para o usuário.
